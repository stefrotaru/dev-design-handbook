# Dev Design Handbook

A personal knowledge base for UI/UX fundamentals, built by a frontend developer 
learning to think like a designer.

## What's This?

Notes, principles and exercises distilled from design resources — primarily 
*Design for Developers* by Stephanie Stimac but not only — organized for quick reference 
and practical application.

## Topics

- **Layout & Composition** — grids, spacing, visual hierarchy
- **Typography** — choosing and pairing typefaces, readability
- **Color** — theory, accessibility, palettes
- **UI Patterns** — common components and when to use them
- **Wireframing** — from sketch to mockup

## Resources

Curated tools, inspiration sites and further reading.
